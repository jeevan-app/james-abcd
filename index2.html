Vaaradhi Form

This project is a static web page developed for Vaaradhi Farms, designed to highlight the farm’s offerings and create a visually appealing online presence. The website is built using HTML, CSS, and JavaScript, focusing on showcasing the beauty and sustainability of farm life. Although it lacks backend functionalities, the web page features a clean, user-friendly interface with responsive design for an optimal experience on various devices.
