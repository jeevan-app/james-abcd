Project Overview

This project is a static website built as a replica of the popular movie and event booking platform, BookMyShow. It was developed using HTML, CSS, and JavaScript to create an aesthetically pleasing, responsive, and user-friendly interface. The project focuses on front-end design and simulates key functionalities of an online booking system without dynamic backend interactions.
